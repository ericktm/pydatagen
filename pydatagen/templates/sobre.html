Pydatagen - Gerador de registros para banco de dados

O Pydatagen é uma ferramenta em desenvolvimento, mantida por Erick Roberto de Oliveira. A proposta do software é gerar scripts de inserção automática de registros relevantes(similares à um ambiente de produção).

Sobre o autor

Erick Roberto de Oliveira é formando em Sistemas para Intenet no Instituto Federal de Mato Grosso do Sul. Trabalha na área de Qualidade de software. Já se envolveu com as seguintes tecnologias:

  • HTML(XHTML,HTML4,HTML5)
  • Javascript(Jquery, ExtJS Framework)
  • Python(SqlAlchemy, Django, Web2py, PyQT4)
  • PHP(Zend Framework, Cake PHP, Symphony, Yii)
  • Linux(Apache, Nginx, Ossec, BindDNS, FTP Server, Firewall)
  • Java (Android, J2EE)
